# Helper Scripts

This directory contains utility scripts for specific tasks. All scripts support the `--help` flag for detailed usage information.

## fetch_protos.sh

This script is used to fetch the proto files from a specific branch.

## generate_docs.sh

Generates project documentation using Sphinx. Creates a temporary virtual environment, installs documentation dependencies, builds the HTML docs, and cleans up.

Output is saved to `docs/build`.

## update_version.sh

Updates SDK, Ankaios, and API versions across the project. Supports updating versions individually or all at once.

## install_ankaios.sh

This script provides multiple ways to install ankaios, from an official release to the executables generated by a github action of a specific branch. Use the `--help` for more details.
